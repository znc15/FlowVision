# 接入外部 MCP 服务器

FlowVision 支持接入外部 MCP 服务器来扩展 AI 能力。

## 支持的传输类型

| 类型 | 说明 | 适用场景 |
| ---- | ---- | -------- |
| **Stdio** | 通过子进程启动 MCP 服务器 | Python/Node.js MCP 服务器 |
| **SSE** | Server-Sent Events 连接 | 远程 MCP 服务器 |
| **HTTP** | Streamable HTTP 连接 | MCP 协议最新传输方式 |

## 添加 MCP 服务器

### 通过界面添加

1. 打开 **设置 → MCP 服务**
2. 点击 **添加服务器**
3. 填写服务器信息：
   - 名称：显示名称
   - 传输类型：选择 stdio / SSE / HTTP
   - 根据传输类型填写命令/URL
   - 环境变量（stdio 模式）
4. 点击 **添加**

### Stdio 传输示例

以添加 GrokSearch 为例：

| 字段 | 值 |
| ---- | -- |
| 名称 | GrokSearch |
| 传输类型 | Stdio |
| 命令 | `uvx` |
| 参数 | `--from git+https://github.com/GuDaStudio/GrokSearch@grok-with-tavily grok-search` |
| 环境变量 | `GROK_API_URL=...` 和 `GROK_API_KEY=...` |

### SSE 传输示例

| 字段 | 值 |
| ---- | -- |
| 名称 | 远程搜索服务 |
| 传输类型 | SSE |
| URL | `http://localhost:3000/mcp/sse` |

## 管理服务器

在 **设置 → MCP 服务** 面板中可以：

- **连接/断开** 服务器
- **测试** 连接是否正常
- **编辑** 服务器配置
- **删除** 服务器
- 查看已连接服务器的**工具列表**

## 在 AI 对话中使用

连接 MCP 服务器后，其提供的工具可通过以下方式使用：

1. **联网搜索**：如果服务器提供 `web_search` 工具，点击输入框旁的搜索按钮启用
2. **API 调用**：通过后端 API 直接调用工具
3. **自动集成**：AI 对话会自动检测并使用可用的 MCP 工具

## API 管理

也可以通过 REST API 管理外部 MCP 服务器：

```bash
# 添加服务器
curl -X POST http://localhost:3001/api/mcp/servers \
  -H "Content-Type: application/json" \
  -d '{
    "id": "my-server",
    "name": "自定义搜索",
    "transport": "stdio",
    "stdio": {
      "command": "node",
      "args": ["path/to/server.js"],
      "env": {"API_KEY": "xxx"}
    },
    "enabled": true
  }'

# 连接服务器
curl -X POST http://localhost:3001/api/mcp/servers/my-server/connect

# 调用工具
curl -X POST http://localhost:3001/api/mcp/call-tool \
  -H "Content-Type: application/json" \
  -d '{"serverId": "my-server", "toolName": "search", "args": {"query": "test"}}'
```
