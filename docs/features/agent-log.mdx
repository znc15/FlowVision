# Agent 日志

FlowVision 提供详细的 Agent 日志系统，记录所有 AI 操作的完整过程。

## 日志类型

| 级别 | 图标 | 说明 |
| ---- | ---- | ---- |
| `info` | 🔵 | 信息类操作（发起请求、开始分析） |
| `success` | 🟢 | 操作成功完成 |
| `warn` | 🟡 | 警告（非致命问题） |
| `error` | 🔴 | 操作失败 |

## 日志字段

每条日志包含以下信息：

| 字段 | 说明 |
| ---- | ---- |
| 来源 | 操作来源模块（AI分析、AI请求、GitHub导入等） |
| 消息 | 操作描述 |
| 状态 | `等待` / `执行中` / `完成` / `失败` / `取消` |
| 步骤 | 当前执行步骤及编号（如 "2/5"） |
| 耗时 | 操作耗时（毫秒） |
| 标签 | 分类标签（如 AI生成、项目分析） |
| 详情 | 可展开的结构化详细信息 |
| 指标 | 性能指标（如节点数、技术栈数量） |

## 查看日志

在右侧边栏切换到 **Agent 日志** 面板，可以：
- 查看所有操作日志列表
- 点击任意日志查看详细信息
- 按级别、来源筛选
- 清空日志

## 日志存储

- 日志存储在浏览器 localStorage 中
- 最多保留 500 条（超出自动淘汰旧记录）
- 刷新页面后日志保留
